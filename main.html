Vulnsense: Detecting Vulnerability in Smart Contracts

This is a web application for detecting vulnerabilities in smart contracts. The web application using a tool which is presented in this paper.

The tool detects 2 types of vulnerabilities, Reentrancy and Arithmetic, in smart contracts written in the Solidity language and deployed on the Ethereum Blockchain.

This web application consists of 2 functions:

1. Upload File: Allows users to upload a .sol or .txt file containing the source code of the contract.
2. Enter Address: The address of the contract deployed on Ethereum.

The application will return the result indicating whether the contract contains either of the two vulnerabilities, Reentrancy or Arithmetic, or not.
